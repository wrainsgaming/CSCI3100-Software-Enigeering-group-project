Objective
  The focus of this project is to offer a platform for CUHK students to get familiar with
stock markets. Stock discussion forum aims to allows users to share their ideas for
further discussion. Stock inquiry allows users to check details of a stock. Stock market
analysis is designed for users to know more about stocks. Stock market simulator
provides a virtual environment for users to get used to the operation of stock
markets. 

How to Test the Project:

Execute git clone https://github.com/wrainsgaming/CSCI3100-Software-Enigeering-group-project in terminal

Go to execute node index.js and chatroom.js

Visit http://localhost:5000/login to the main page of the web app

For the admin, use "admin" as account and "admin" as password to login
